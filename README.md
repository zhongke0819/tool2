# 视频下载器

一个支持 TikTok 和 YouTube 视频下载的网页应用。

## 功能特点

- TikTok 视频下载（无水印）
- YouTube 视频搜索和下载
- 响应式设计，支持移动端
- 多语言支持

## 技术栈

- HTML5
- CSS3
- JavaScript
- Python (用于本地服务器)

## 本地运行

1. 克隆仓库
```bash
git clone [你的仓库地址]
```

2. 进入项目目录
```bash
cd [项目目录]
```

3. 启动本地服务器
```bash
python -m http.server 8000
```

4. 在浏览器中访问
```
http://localhost:8000/public/
```
